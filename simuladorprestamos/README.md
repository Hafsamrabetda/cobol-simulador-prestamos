# Simulador de Préstamo en COBOL

Proyecto personal desarrollado en COBOL como parte de mi aprendizaje autodidacta del lenguaje.

## Descripción
El programa simula un préstamo sencillo solicitando el importe, el interés anual y el número de años, y calcula el total a pagar y la cuota mensual. El objetivo del proyecto es practicar la lógica de negocio básica, el uso de variables, cálculos y la salida por pantalla en un entorno COBOL.

## Tecnologías
- COBOL

## Ejecución
El programa se puede ejecutar en compiladores COBOL online como JDoodle, introduciendo los valores solicitados por consola.
